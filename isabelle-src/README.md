# Isabelle/ZF sources (vendored, read-only reference)

These `.thy` files are the **Isabelle/ZF** object-logic sources that this repo
ports to Lambdapi. They are the source of truth for both statement fidelity and
completeness audits (`tools/audit.py`).

Provenance: the Isabelle distribution (`src/ZF`), distributed under the
Isabelle/BSD-style licence (see <https://isabelle.in.tum.de>). They are included
here unmodified, purely as reference material for the port — do not edit them.
